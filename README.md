## Zen Chen

Building **streaming micropayments for AI agents** on Circle's Arc.

### Now

[**Arc402**](https://github.com/Ccheh/arc402) -- agents pre-deposit USDC once into an on-chain escrow, then sign cheap off-chain EIP-712 claims per API call. Sub-cent per-call billing, zero on-chain overhead per request. The streaming-payment layer that complements Circle's [ERC-8183](https://docs.arc.network/arc/tutorials/create-your-first-erc-8183-job) (discrete jobs) and [ERC-8004](https://docs.arc.network/arc/tutorials/register-your-first-ai-agent) (agent identity).

Live on Arc Testnet -- [PaymentEscrow `0x55aFA5...2a98`](https://testnet.arcscan.app/address/0x55aFA5Cf28B98DD6DC550F15c075F46B5eaf2a98) -- full SDK and end-to-end demo.

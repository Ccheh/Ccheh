## Zen Chen

Building **Cadence** -- streaming USDC micropayments for AI agents on Circle's Arc.

### Now

[**Cadence**](https://github.com/Ccheh/arc402) (Arc402 protocol) -- agents pre-deposit USDC once into an on-chain escrow, then sign cheap off-chain EIP-712 claims per API call. Sub-cent per-call billing, zero on-chain overhead per request. The streaming-payment layer that complements Circle's [ERC-8183](https://docs.arc.network/arc/tutorials/create-your-first-erc-8183-job) (discrete jobs) and [ERC-8004](https://docs.arc.network/arc/tutorials/register-your-first-ai-agent) (agent identity).

Live on Arc Testnet -- [PaymentEscrow v2 `0xc95b1b...82f8d`](https://testnet.arcscan.app/address/0xc95b1b20f91901206ba3ea94bbc7313e7cd82f8d) -- 30 tests passing, 20-claim batch verified, 5/5 adversarial attacks blocked.
